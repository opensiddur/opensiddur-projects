# opensiddur-projects

Jewish Liturgy sources in TEI, suitable for use with the Open Siddur compiler/exporter in [opensiddur-ai](https://github.com/opensiddur/opensiddur-ai). Original sources are in the [sourcetexts](https://github.com/opensiddur/sourcetexts) repository.

Copyright license information for each file is in the TEI header's licence element in each file.
